### How to play this game
This is a multi-user game and each side is controlled by 1 user. 
User 1 controlling the left side panel uses "W" key to go up and "S" key to go down.
User 2 controlling the right side panel uses "Up" and "Down" keys to move the panel up
and down. Both users must not let the ball pass each side and have the ball bounce back from 
their panel. If failed to do so, the opponent gets a point. 
The user with the highest score wins.
